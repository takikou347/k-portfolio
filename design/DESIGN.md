# こくばん デザインシステム

コンセプト: 「放課後の黒板」。深緑の黒板にチョークで落書きし、画用紙の付箋をマグネットで
留める。他の人のカーソルは名前入りマグネットとして盤上を動く。
汎用ホワイトボードツール (FigJam/Miro) のクローン見た目にはしない。

## タイポグラフィ

- 付箋・カーソル名札・見出し: Yusei Magic (チョーク/手書きの趣。Google Fonts)
- UI (ツールバー・ダイアログ): Zen Kaku Gothic New (400 / 500)
- 座標・人数などのデータ: IBM Plex Mono (500)

## 画面構成

- 全画面が黒板。上下に細い木枠 (--color-frame)。下枠は「チョーク受け」として
  ツールバーを兼ねる: チョーク4色 (実物のチョーク型ボタン)、黒板消し、付箋、リアクション
- 右上に現在の接続人数を mono で小さく表示 (例: "5 人が見ています")
- パン: ドラッグ (手のひらツール or Space)、ズーム: ホイール/ピンチ (50%〜200%)
- 初回入室時に名前入力 (2〜8文字)。名前とカーソル色はランダム提案 + 変更可

## シグネチャ要素 1: チョークの描き味

- ストロークはかすれ・粒状感のある線 (canvas 2D で低透明度の重ね描き + わずかなジッター)
- 描いている間、線の先端からチョークの粉 (小さな粒子) がわずかに落ちる
- 黒板消しはスタンプ型: なぞった範囲のストロークが「拭き跡」(薄く白が残る smudge) と共に消える

## シグネチャ要素 2: マグネットと画用紙

- 付箋 = 画用紙: わずかに不揃いな角 (border-radius を四隅で変える)、上辺中央に赤マグネット
- ドラッグ中は影が深くなり 1.5deg 傾く。ドロップでピタッと吸着 (scale 1.02 -> 1)
- 他人のカーソル = 小さな丸マグネット + 手書き風の名前ラベル。移動は 120ms の ease で追従

## リアクション

- ツールバーの「👏 ✨ 💮 ❤️」をタップ → 自分のカーソル位置で絵文字が浮かび上がり、
  チョークの粉がふわっと舞って 1.2 秒で消える。全員の画面に表示。連打可 (秒 3 回まで)

## 品質基準

- モバイル: ツールバーは下部固定のまま。ピンチズーム・1本指パン・付箋ドラッグが両立すること
- prefers-reduced-motion: 粉・吸着・浮遊アニメーションを無効化 (機能は維持)
- フォーカスリングは chalk-yellow の 2px アウトライン
- 初見で操作が分かること: 空の黒板に薄いチョークで「じゆうに かいてね」のプレースホルダー
  (誰かが最初のストロークを描いたら消える)

## レスポンシブ / 入力系統 (必須仕様)

ブレークポイント: SP ≤639px / Tablet 640〜1023px / Desktop ≥1024px

### 入力
- Desktop: ホイール = カーソル中心ズーム (50〜200%)、Space + ドラッグ = パン、
  Ctrl/Cmd+Z = 自分のストロークの取り消し
- タッチ (SP/Tablet): 1 本指 = 選択中ツールの操作 (チョークなら描画)、
  2 本指 = パン + ピンチズーム。手のひらツールは置かない (2本指で代替)
- ズームコントロール: 右下に + / − / 100% リセットの縦ボタン (タッチでも使える保険)。
  現在倍率を mono で表示

### ツールバー (チョーク受け)
- Desktop: 下部中央に一列
- SP: 下部固定・横スクロール可。タップ領域は最小 44px。選択中チョークは受けから浮き上がる
- 取り消しボタン (自分の直近ストロークのみ対象) をチョークの隣に置く

### 付箋の編集
- Desktop: ダブルクリックでその場で textarea 編集。Esc で確定
- SP: タップで下からボトムシート (テキスト・色・削除をまとめる)。ドラッグは長押しで開始
  (スクロール/パンとの誤爆防止)

### 入室・状態表示
- 名前入力: Desktop は中央モーダル、SP はボトムシート。ランダム名を初期表示し 1 タップで入れる
- 再接続中: 上部にチョーク文字のバナー「つなぎなおしています…」。復帰で 1 秒表示「つながりました」
- 満席時: 黒板は読み取り専用で表示し、バナー「いま満席です。しばらくしてから来てね」
- 人数表示をタップ/クリックすると参加者の名前一覧がポップオーバーで見える

### 検証基準
- iPhone SE (375px)、iPad (768px)、FHD (1920px) の 3 幅で全操作を確認する
- SP で「2本指パン中に誤描画しない」「付箋ドラッグとパンが衝突しない」ことを必ず確認
