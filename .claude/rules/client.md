# クライアント (React SPA) 規約

- 状態は zustand 一本。ボード状態は shared/ の reducer をそのまま使い、クライアント独自の状態遷移を書かない
- 自分の操作は楽観的に即時適用し、同じ op をサーバーへ送る。サーバーからの自分の op エコーは無視する
- 描画: チョークストロークは <canvas> (2D) に描画、付箋・カーソルは DOM 要素。混在させない
- WebSocket 再接続: 切断時は指数バックオフで自動再接続し、再接続時にスナップショットを取り直す
- カーソル送信は 80ms スロットル。ストローク中の点は 16ms ごとにバッチして送る
- パン/ズームは transform で実装し、座標系は「ボード座標」で統一 (画面座標をイベント処理の外に漏らさない)
- prefers-reduced-motion で装飾アニメーションを無効化する
- インタラクティブ要素は button/a。div + onClick 禁止。フォーカスリングを消さない
