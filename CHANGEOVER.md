# サービス差し替えプロンプト (Claude Code に貼り付ける)

用途: リポジトリに旧モック (進行ボード等) の実装が既にある場合に使う。
空のリポジトリにハーネスを置いた直後なら、これは不要で KICKOFF.md をそのまま貼ればよい。

---

このリポジトリのサービスをリアルタイム共有黒板「こくばん」に差し替えます。
CLAUDE.md / KICKOFF.md / design/DESIGN.md / .claude/rules/ は既に新サービスの内容に更新済みです。
ハーネスの hooks / agents / workflows は変更しないでください。

まず計画モードで移行計画 (流用するもの・捨てるものの棚卸しを含む) を提示し、
承認後に KICKOFF.md のスライス順で自律的に進めてください。
最後に grep で旧サービスの痕跡 (旧テーブル・旧コンポーネント・旧トークン) がゼロ件であることを
確認してください。完了条件は KICKOFF.md と同一です。
